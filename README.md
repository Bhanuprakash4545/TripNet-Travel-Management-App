# ✈️ TripNet - Travel Management App

TripNet is a dynamic Android application designed to help travel agencies efficiently manage their daily operations.  
The application allows travel businesses to track bookings, manage expenses, record payments, and automatically calculate profit or loss using real-time data synchronization.

---

## 🚀 Features

- Trip booking management
- Expense tracking system
- Payment transaction tracking
- Automatic profit and loss calculation
- Real-time data synchronization using Firebase
- Dynamic dashboard for operations overview

---

## 🛠 Tech Stack

- Kotlin (Android Development)
- Firebase Cloud Firestore
- Android Studio
- Material UI Components

---

## 📌 Key Functionalities

- Add and manage travel bookings
- Record trip-related expenses
- Track customer payments
- Automatically calculate operational profit or loss
- View business data in real-time
